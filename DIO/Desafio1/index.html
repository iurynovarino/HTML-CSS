Estrutura básica do HTML

HTML é uma linguagem de marcação usada para criar e estruturar conteúdo para Web. Basicamente é como um esqueleto de uma página Web, onde você define os elementos que compõem a página, como textos, imagens, links e vídeos.

A instrução !DOCTYPE deve ser sempre a primeira a aparecer em uma página HTML para indicar ao browser qual versão da linguagem usada.

A tag HTML, delimita o documento. Sendo assim, todas as demais tags da página devem estar nesse espaço

abertura e fechamento da tag head, que define o cabeçalho do documento. O conteúdo nesse espaço não é visível no browser, mas contém instruções sobre seu conteúdo e comportamento. Dentro dessa tag, por exemplo, podem ser inseridas folhas de estilo e scripts.

abertura e fechamento da tag body, marcando o espaço no qual deve estar contido o conteúdo visual da página. As demais tags que representam texto, botões etc. devem ser adicionadas nesse intervalo.

Atributos básicos

O atributo de uma tag é uma propriedade que pode ser adicionada a um elemento HTML para fornecer informações adicionais ou modificar o seu comportamento. A sintaxe para adicionar um atributo a uma tag é incluí-lo dentro do par de tags da seguinte maneira: <tag atributo="valor do atributo">

Existem diversos atributos disponíveis para diferentes elementos do HTML, como class, id, href, src, width e etc.
